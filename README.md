# my-screen-terminal
MyScreen Terminal

## Dependencies

To run this project you need to have:

* Java 1.7+
* [Maven](https://maven.apache.org/download.cgi)

## Setup the project

### Building

```
$ mvn package
```

### Running

```
$ java -cp target/my-screen-terminal-<version>.jar com.myscreen.terminal.MyScreenTerminal
```
